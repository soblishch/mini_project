### Вибір компанії та отримання даних
Для проєкту ми обрали українську аграрну компанію **Kernеl**, одну з найбільших виробників та експортерів соняшникової олії у світі.
Бо вважаємо, що інвестиції в аграрний сектор в Україні є рентабельними та підтримують нашу економіку.

Щоб отримати дані ми підключили бібліотеку **yfinance** та використали `ticker = "KER.WA"`.

Витягнули дані за кожний день протягом року:  `data = yf.download(ticker, start='2025-07-19', end='2026-07-19', interval="1d")`

Після цього отримали такий DataFrame
```text
Price           Close       High        Low       Open Volume
Ticker         KER.WA     KER.WA     KER.WA     KER.WA KER.WA
Date                                                         
2025-07-21  18.139999  18.580000  17.900000  18.000000   7026
2025-07-22  18.100000  18.139999  17.980000  18.139999   3071
```

Yfinance робиль багаторівневий header, тож для зручності ми забрали рядок з KER.WA : `data.columns = data.columns.droplevel(1)`

Команда звертається до заголовка таблиці, формуючи:
```text
MultiIndex([( 'Close', 'KER.WA'),
            (  'High', 'KER.WA'),
            (   'Low', 'KER.WA'),
            (  'Open', 'KER.WA'),
            ('Volume', 'KER.WA')],
           names=['Price', 'Ticker'])
```
та видаляє другий стовпець

Потім ми видаляємо рядки зі значенням N/A, щоб не було помилок при обчисленнях: `data = data.dropna()`
